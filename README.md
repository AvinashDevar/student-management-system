# Student Management System

A web-based Student Management System developed using JSP, Servlets, JDBC, and MySQL. The application allows students to register, log in, manage their profiles, and access course-related information.

## Features
- Student Registration
- Student Login & Authentication
- Profile Management
- Course Management
- Admin Dashboard
- Database Connectivity using JDBC

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML5
- CSS3
- JavaScript
- Apache Tomcat

## Project Structure

```text
java/
├── controller/
├── dao/
├── dto/
└── jpUtil/

resources/
└── persistence.xml/

webapp/
└── WEB-INF/
```

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone <https://github.com/AvinashDevar/student-management-system>
   ```

2. Import the project into Eclipse/STS.

3. Configure the MySQL database and update database credentials.

4. Deploy the project on Apache Tomcat.

5. Open the application in your browser.
